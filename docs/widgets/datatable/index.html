DataTable

NOTE: New since v0.5.3.

WARNING: While this Widget is available in Javascript-disabled Browsers, it can result in Cumulative Layout Shift.

DataTable is a subset of Table Widget which automatically handles formatting tabular data into HTML.

<script>

        {
            text: "First Name",
            key: "first_name",
        },

        {
            text: "Last Name",
            key: "last_name",
        },

            text: "Occupation",
            key: "occupation",
            sorting: true,

                // Instead of alphabetizing, we're sorting by length of

DataTable

Name Description Types
columns Configures metadata on how DataTable should renders the tabular data.
{key: keyof T, text: string, sorting?: boolean, sorting_algorithm?: (a: T[keyof T], b: T[keyof T]) => number}[]
rows Sets the tabular data that DataTable is to render.
T[]
page Sets the current page of the pagination.
1 (DEFAULT) number string
paginate Enables DataTable to split the tabular data into paged views.
boolean
paging Sets how many rows should appear per page.
5 (DEFAULT) number string
sorting Sets which row member that DataTable is currently sorting by.
keyof T
sorting_mode Sets which direction DataTable is sorting the row member by.
ascending descending
searching Sets the current search query that DataTable is using to filter the tabular data.
string
searching_algorithm Optional custom searching filter. By default, all row members are lowercased and fuzzy searched.
(row: T) => boolean
palette Alters the displayed color scheme.
auto inverse inherit accent dark light alert affirmative negative
sizing Alters the overall spacing / sizing.
tiny small medium large huge
variation Alters how the DataTable renders with various additions.
borders stripes
