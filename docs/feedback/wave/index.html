Wave

NOTE: New since v0.2.5.

DEPRECATED: This feature will be removed for <Ellipsis animation="bounce" iterations="5"> in v0.6.0 due to expanded functionality.

Wave is an indicator used to tell the end-user that the Web Application is loading something in the background.

<script>
    import {Hero, Wave} from "@kahi-ui/framework";
</script>

<Hero.Container margin_top="medium">
    <Hero.Header>Loading</Hero.Header>
    <Hero.Footer>
        <Wave />
    </Hero.Footer>
</Hero.Container>

Imports

<script>
    import {Wave} from "@kahi-ui/framework";
</script>

Palette

You can change the color palette of the Wave via the palette property.

<script>
    import {
        Stack,
        Text,
        Wave,
    } from "@kahi-ui/framework";
</script>

<Stack
    class="wave-palette"
    spacing="medium"
    orientation="horizontal"
    variation="wrap"
>
    <div>
        <Text is="strong">DEFAULT</Text>
        <Wave />
    </div>

    <div>
        <Text is="strong">ACCENT</Text>
        <Wave palette="accent" />
    </div>

    <div>
        <Text is="strong">DARK</Text>
        <Wave palette="dark" />
    </div>

    <div>
        <Text is="strong">LIGHT</Text>
        <Wave palette="light" />
    </div>

    <div>
        <Text is="strong">ALERT</Text>
        <Wave palette="alert" />
    </div>

    <div>
        <Text is="strong">AFFIRMATIVE</Text>
        <Wave palette="affirmative" />
    </div>

    <div>
        <Text is="strong">NEGATIVE</Text>
        <Wave palette="negative" />
    </div>
</Stack>

<style>
    :global(.wave-palette .wave) {
        display: block;
    }
</style>

API Reference

Wave

Name Description Types
palette Alters the displayed color scheme.
auto inverse inherit accent dark light alert affirmative negative
size Renders the Wave at a different sizes.
tiny small medium large huge

Wave

Name Description Types
default Default unnamed slot.
{}

Wave

Name Description Types
--wave-border-radius Controls the border-radius used for Wave dots.
<length> <percentage>
--wave-font-size Controls the font-size used, which the Wave dots respect for width / height.
<length> <percentage>
--wave-content-second-animation-delay Controls how long of a delay until the second Wave dot is bounced in sequence.
<time>
--wave-content-third-animation-delay Controls how long of a delay until the third Wave dot is bounced in sequence.
<time>
--wave-content-fourth-animation-delay Controls how long of a delay until the fourth Wave dot is bounced in sequence.
<time>
--wave-content-fifth-animation-delay Controls how long of a delay until the fifth Wave dot is bounced in sequence.
<time>
--wave-bounce-top-animation-function Controls the animation timing function switched to when a Wave dot reaches the peak of the bounce.
<easing-function>
--wave-bounce-bottom-animation-function Controls the animation timing function switched to when a Wave dot reaches the base of the bounce.
<easing-function>
