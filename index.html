Easy to use Svelte UI for
rapid prototyping
Kahi UI is a composable and flexible design library for Svelte that allows you to go from 0 to 60 in no time.
Source Getting Started

Straight-forward. Minimal Markup.

Develop more UI without writing essays worth of markup.

Features expected of a modern UI.

Opinonated when needed. Flexible when wanted.

Composable

Designed with Composability in mind, author new Components easily.

Staticly Typed

Created using Typescript, so a pleasent IDE experience comes out-of-box.

Dark Mode

Supports automatic Dark Mode, or your choice from the built-in palette.

Progressively Enhanced

Most logic is embedded within HTML / CSS, with Javascript enhancing the UX.
